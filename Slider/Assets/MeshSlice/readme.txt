Fix compilation error.

1) The asset uses DOTween (HOTWeen v2).
Download and setup it https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676

2) If you use Unity 2019, you could face with Library/PackageCache/com.unity.package-manager-ui errors
- Close Unity
- Remove all folders except Assets and ProjectSettings
- Restart Unity
